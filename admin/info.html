StarAzure - HomeRun - Premium Joomla Theme

StarAzure HomeRun is a super fast Joomla template built using Bootstrap 4 and Google Material Icons. The theme comes with a built in customizer which makes it very easy to modify your site without programming.

Choose from a variety of colors, icons and web elements. This awesome theme is so light weight and will rank your page high on Google.

Out of the box, this theme comes with only 1 javascript file and 1 css file that includes Bootstrap 4. Thats it ! Now you know how we make our theme fast and furious !!
